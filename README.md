### Descripción
El proyecto consiste en una Máquina de Turing que permite ingresar la descripción formal de la máquina y que el programa pueda simular la entrada configurada y su implementación. Como resultado se da una configuración final sobre la cinta y luego se indica si el resultado final es aceptado o no. Para este proyecto se utilizó Python.

### Ejecución del programa
Para poder ejecutar el programa se utilizan los siguientes comandos:
+ ``` python main.py alterern.yaml``` ejecuta la máquina de turing alteradora.
+ ``` python main.py alterern2.yaml``` ejecuta la máquina de turing alteradora con otro ejemplo.
+ ``` python main.py recognizer.yaml``` ejecuta la máquina de turing reconocedora.
